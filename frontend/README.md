# Frontend для тестирования API

Простой одностраничный фронтенд для тестирования всех API endpoints.

## Запуск

Просто откройте `index.html` в браузере:

```bash
# Через Python
cd frontend
python3 -m http.server 3000

# Или просто откройте файл в браузере
open index.html
```

Затем откройте в браузере: `http://localhost:3000`

## Функциональность

### Аутентификация
- Регистрация пользователя (viewer/creator)
- Вход в систему
- Отображение информации о текущем пользователе
- Сохранение токена в localStorage

### Управление постами
- Создание поста (для креаторов)
- Получение поста по ID
- Список всех постов

### Лента
- Получение персонализированной ленты
- Фильтрация ленты по категории

### Кошелёк
- Просмотр баланса
- Пополнение баланса
- Покупка поста
- История транзакций

### Аналитика (для креаторов)
- Статистика креатора
- Статистика по конкретному посту
- Просмотр доходов

## Использование

1. Зарегистрируйтесь или войдите
2. После входа отобразятся все доступные секции
3. Используйте формы для отправки запросов к API
4. Результаты отображаются в формате JSON
